使用Vultr搭建SS服务器翻墙

发表于 2020-02-08 更新于 2020-02-09
本文字数： 1.9k 阅读时长 ≈ 3 分钟

本篇文章，我将使用Vultr服务商提供的VPS服务器，搭建 Shadowrocks 并开启 BBR 加速上网功能

特别说明：本次博客中提到的软件均已进行备份，如原链接失效，可查看 备份页面

之前一直使用 谷歌访问助手 翻墙，虽然比较稳定，但是它限制访问 YouTube 等网站；
我还尝试过使用像 顺风云、加多宝 这样的节点订阅，但这样的节点不太稳定，访问速度也不太快；
尝试使用过 lncn.org 这样的节点网站，虽然利用无法订阅的特点减少了阻塞情况，但几乎每次使用都需要去网站更新节点，很不方便；
因此，本次我将尝试使用自建的VPS服务器，搭建 Shadowrocks 服务，通过BBR实现网络加速功能

一、优惠购买vultr

在查阅一些资料后，觉得 Vultr 比较合适SS服务器的搭建，所以首先去购买一台VPS服务器资源

注册账户之后，选择需要的服务器类型，可以参考图1我的配置自行选择

图1 服务器选择

注意：日本的IP很可能被封，所以不建议使用日本的服务器，建议使用新加坡的服务器

点击右下角 Deploy Now 进行购买，进入充值界面

图2 充值页面

目前 Vultr 官网 的活动是：充值10美元，赠送100美元
最低充值额度是10美元，可以使用支付宝支付
另外，通过此链接注册支付，还会额外赠送25美元（给自己打个广告😜）

图3 活动详情

注册链接: https://www.vultr.com/?ref=8426042-6G

当你购买完成，等待一段时间让服务器完成镜像安装，就可以进行 Shadowrocks 配置了

四、配置SSR客户端

  • iOS 用户

iOS 的 Shadowsocks 客户端在国区被封了，美区也需要购买才能使用，购买方式可以查看这篇 GitHub博文

Android 就比较幸运了，有很多可用的软件
